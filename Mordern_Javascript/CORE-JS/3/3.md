`let` 키워드로 변수를 선언할 수 있다.
```js
let message;
```

변수는 값을 바꿀 수 있다.
```js
message = 'Hello';
```

한 줄에 여러 개의 변수를 선언할 수도 있다.

```js
let user = 'John', age = 25;
```

`const` 키워드로 상수를 선언할 수 있다.
```js
const MY_BIRTHDAY = '2005.05.25'
```

상수는 최초 선언 이후 수정할 시, 에러가 발생한다.
```js
const MY_AGE = 20;
MY_AGE = 19; // Error 발생!!
```
